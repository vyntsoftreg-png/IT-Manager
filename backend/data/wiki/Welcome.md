# Welcome to IT Knowledge Base

This is your internal Wiki system. You can use it to store:

- **Procedures**: SOPs, Setup guides.
- **Troubleshooting**: Common errors and fixes.
- **Network Diagrams**: IP lists, VLAN configs.

## How to use
- **Create**: Click the + icon in the sidebar to create a new page or folder.
- **Edit**: Select a page and start typing in the editor.
- **Markdown**: This system supports standard *Markdown* formatting.

### Example Code Block

# Update server
sudo apt update && sudo apt upgrade
